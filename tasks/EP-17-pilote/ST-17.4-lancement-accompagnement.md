## Contexte business

Le pilote (10-12 semaines avec 3-5 stagiaires) est la phase de validation produit-marché-utilisateur. Sans accompagnement actif les 3 premières semaines, le risque de drop-off est maximal (premières frictions, première soumission, premier blocage). Cette story définit le rituel d'accompagnement, les métriques surveillées, le canal de support.

## Description fonctionnelle

Kick-off 1h en visio. Canal Discord/Slack pilote ouvert. Mohamed disponible asynchroniquement (réponse < 4h en heures de bureau). Office hours hebdo (1h). Suivi quotidien des soumissions + alertes via le dashboard formateur. Post-week 1, 2, 3 : entretiens individuels 15 min par stagiaire (feedback).

Métriques surveillées : taux d'engagement (DAU), taux de soumission, taux de validation harnais 1ᵉ essai, alertes ouvertes, temps de résolution.

## Critères d'acceptation (Gherkin)

**Scénario 1 — Kick-off**

```gherkin
Given les stagiaires sont onboardés
When le kick-off 1h a lieu
Then chacun présente son contexte
And Mohamed présente le cursus + outil + règles
And la cohorte se sent alignée (sondage rapide post-meeting)
```

**Scénario 2 — Première semaine support actif**

```gherkin
Given le pilote est en semaine 1
When un stagiaire pose une question Discord
Then Mohamed répond < 4h en heures de bureau
And un message journalier "aide pingue" est posté
```

**Scénario 3 — Office hours**

```gherkin
Given le mardi 18h
When Mohamed lance l'office hours visio
Then au moins 2 stagiaires participent
And les questions sont logées dans un faq
```

## Cas limites à traiter

- **Stagiaire décroche** : email perso de Mohamed, ne pas blamer, comprendre.
- **Bug critique découvert pilote en cours** : hotfix < 24h.
- **Soumissions massives le même jour** : Mohamed prépare batch de review.
- **Drama interpersonnel cohorte** : intervention Mohamed (médiation privée).
- **Capstone impossible pour 1 stagiaire** : revoir scope, ne pas humilier.

## Non-goals

- Pas d'accompagnement payant.
- Pas d'engagement de résultat (Cursus = outil, pas contrat).
- Pas de pression sur les stagiaires à finir vite.
