## Contexte business

roadmap.sh fournit une excellente structure pédagogique (frontend, backend, devops, cybersec, AI...). On veut s'en inspirer pour générer le squelette d'un cursus en quelques clics. **Recherche juridique préalable obligatoire** : roadmap.sh est sous CC BY-SA 4.0 — attribution obligatoire.

Valeur livrée : génération de cursus de qualité en 30s au lieu de 2h, avec contenu validé par la communauté roadmap.sh.

## Description fonctionnelle

Modale "Importer depuis roadmap.sh" qui propose une liste de roadmaps disponibles (frontend, backend, devops, cybersec, AI, etc.), importe la structure (concepts ordonnés) comme modules vides. Lien vers le concept conservé en métadonnée. Attribution conforme licence.

## Critères d'acceptation (Gherkin)

**Scénario 1 — Sélection d'une roadmap**

```gherkin
Given un formateur crée un nouveau cursus
When il clique "Importer depuis roadmap.sh"
Then une liste de roadmaps disponibles s'affiche
And il peut filtrer par catégorie
```

**Scénario 2 — Import effectif**

```gherkin
Given le formateur sélectionne "Frontend"
When il confirme l'import
Then la structure (liste ordonnée de concepts) est importée comme modules vides
And chaque module a son titre = nom du concept
And le lien vers le concept sur roadmap.sh est conservé en métadonnée
And une attribution "Inspiré de roadmap.sh" est ajoutée à la description du cursus
```

**Scénario 3 — Source indisponible**

```gherkin
Given roadmap.sh est down
When l'import est tenté
Then un message clair s'affiche "Source indisponible, réessaie plus tard"
And le bouton retry est offert
```

## Cas limites à traiter

- **roadmap.sh down** : message clair, retry possible.
- **Roadmap déprécié** : warning sur la métadonnée.
- **Import sur cursus non vide** : merge ou replace, demandé à l'utilisateur.
- **Cache local périmé** : update auto 1×/mois.
