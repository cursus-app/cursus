## Contexte business

Emails transactionnels critiques (invitation, alerte formateur, validation, capstone, rappel). Sans Resend, on dépend de SMTP fragile et la délivrabilité est médiocre.

Valeur livrée : envoi fiable via Resend, templates responsive en FR/EN, tracking livraison, désabonnement par catégorie.

## Description fonctionnelle

Service `email.ts` avec types stricts. Resend API. Templates Vue Email (preview Storybook). FR (et EN si user EN). Lien désabonnement par catégorie en pied. Webhook Resend pour delivery tracking.

## Critères d'acceptation (Gherkin)

**Scénario 1 — Envoi templaté**

```gherkin
Given un événement requiert un email
When il est déclenché
Then un email responsive est envoyé via Resend
And le template est en FR (et EN si user EN)
And un lien de désabonnement par catégorie est en pied
And la délivrance est trackée (Resend webhook)
```

**Scénario 2 — Templates au MVP**

```gherkin
Given les 5 templates sont implémentés
Then chacun a sa version FR + EN
And chacun a un sujet clair
```

**Scénario 3 — Désabonnement**

```gherkin
Given un user clique sur le lien de désabonnement
When il choisit la catégorie
Then sa préférence est mise à jour
And futurs emails de cette catégorie skipés
```

## Templates au MVP

- Invitation
- Alerte stagiaire (au formateur)
- Livrable validé
- Capstone validé + certificat
- Rappel échéance

## Cas limites à traiter

- **Resend down** : queue + retry exponential.
- **Email bounce** : tracké via webhook, marqué invalid en DB.
- **Spam complaint** : opt-out complet automatique.
- **Render template error** : log + fallback texte brut.

## Considérations sécurité

- DKIM + SPF + DMARC configurés.
- Pas de PII excessive dans les emails.
- Échappement HTML strict (XSS prevent).
- Webhook Resend avec signature HMAC.

## Considérations accessibilité

- Templates avec structure semantique HTML.
- Alt text sur images.
- Liens explicites (pas "cliquez ici").
- Texte brut fallback pour clients sans HTML.

## Considérations performance

- Envoi async (queue Inngest).
- Templates pré-compilés.
